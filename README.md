
 - What is JSX, and why is it used?
    - A syntax that looks like HTML but is written in JavaScript, used to define React components.
 - What is the difference between State and Props?
    - State: Data that a component owns and can change over time.
    - Props: Data passed from a parent component to a child component; read-only.
 - What is the useState hook, and how does it work?
    - A React hook that lets you add state to functional components.
 - How can you share state between components in React?
    - Passing state from a parent to child via props or lifting state to a common parent.
 - How is event handling done in React?
    - Using functions with React events (like onClick) to respond to user actions.
